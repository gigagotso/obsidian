# Away Reasons Icon Redesign

The current Away Reasons icon is confusing, visually weak, and does not clearly communicate its purpose.

Please propose a better icon that represents an agent status change with an associated reason. Consider symbols related to status, availability, transitions, cycles, workflows, or state changes. The icon should be easily recognizable, highly visible, and clearly distinguishable from other status-related icons.

Focus on creating an icon that immediately conveys: "This status changed for a specific reason."
