# DataGrid Header Filters

Implement column-level filters in the DataGrid similar to the filters available in the Conversation Reports table headers.

Each page should expose filter controls relevant to its own columns and available data. The filtering experience should be consistent across the application.
## Requirements

- Add header filters to DataGrid columns where filtering is supported.
- Use filter types appropriate for each column
- Ensure filters are connected to the page's existing filtering logic and API parameters.
- Maintain consistency with the Conversation Reports filtering experience.
- Remove duplicated or page-specific filtering implementations where a shared DataGrid solution can be used.

## Goal

Provide a consistent, reusable column filtering experience across all DataGrid-based pages, similar to the functionality available in Conversation Reports.
